# DESIGN.md - 安定区

## 气质与意象

深夜街角一扇亮着暖光的窗。窗外是深蓝色的夜空和安静的街道，窗内是柔软的灯光和一杯热茶。"安定区"是城市独居青年在情绪低谷时可以躲进来的那个空间——不是诊所，不是工具，而是一个"有人在场"的房间。

## 视觉策略

- **背景**：深邃的夜空渐变，从深靛蓝 (#0f0e2a) 过渡到午夜蓝 (#1a1640)，带有极细微的星点光斑
- **卡片/气泡**：毛玻璃质感（glassmorphism），半透明 + 柔和模糊，让内容"浮"在夜空上
- **图形语言**：所有元素使用大圆角（16px+），无锐角；图标使用线性细描边风格
- **光效**：温暖的琥珀色微光作为点缀，模拟远处路灯或烛光

## 配色方案

| 用途 | 色值 | 意象 |
|------|------|------|
| 主背景 | #0f0e2a → #1a1640 渐变 | 深夜天空 |
| AI气泡 | rgba(255,255,255,0.06) + blur | 月光下的薄雾 |
| 用户气泡 | rgba(196,160,96,0.15) + blur | 窗内透出的暖光 |
| 主文字 | #e8e4f0 | 月光下的白纸 |
| 次要文字 | #9b95ad | 远处的低语 |
| 强调色 | #c4a060 (琥珀金) | 温暖的光点 |
| 危机警示 | #e87170 (柔红) | 关切而非警告 |
| 情绪标签 | 各情绪对应柔和色 | 情绪的自然色彩 |

## 字体排版

- 字体族：PingFang SC / Hiragino Sans GB / Microsoft YaHei（系统默认）
- 对话文字：16px，行高 1.6，保证阅读舒适度
- 标题：轻量（font-weight: 300-400），留白充足
- 整体节奏：宽松、呼吸感强，段落间距大于常规

## 动效与交互

- 消息出现：从下方 12px 处淡入上移，300ms ease-out
- 打字指示器：三个圆点依次柔和脉冲（opacity 0.3→1），周期 1.2s
- 页面切换：整体淡入，400ms
- 悬停反馈：轻微亮度提升（brightness 1.05），无位移
- 缓动曲线：统一使用 cubic-bezier(0.22, 1, 0.36, 1)（柔和减速）

## 设计禁忌

- 禁止使用高饱和荧光色（霓虹蓝、荧光绿等）
- 禁止使用锐角矩形或小圆角（<12px）
- 禁止使用弹跳、抖动、旋转等激烈动画
- 禁止大面积纯白背景（破坏沉浸感）
- 禁止使用红色作为常规警示色（仅在危机模式使用柔红）
- 禁止密集排版，保持"呼吸感"
